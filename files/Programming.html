The best AI resources in programming

GitHub Copilot

An AI-powered tool from GitHub that helps you write code faster. It uses OpenAI's GPT-3 model to provide automatic code suggestions while you type.

Tabnine

An AI-powered tool that assists in code completion. It works with many IDEs such as VS Code, IntelliJ IDEA, and more.

Kite

An AI-powered tool that offers real-time code suggestions as you type in various IDEs.

Codex

A powerful language model from OpenAI specifically trained to understand and generate code. It can assist you in generating code based on textual descriptions.

Replit

An online development platform that integrates AI to provide coding assistance, such as code generation and error correction.

Sourcery (for Python)

An AI-powered tool that helps improve the quality of your Python code.

IntelliCode (by Microsoft)

An AI feature integrated into Visual Studio and Visual Studio Code that provides intelligent suggestions as you write code.

CodeGuru (by Amazon)

An AI-powered tool from Amazon for code review that provides suggestions for performance and security improvements.

AI Pair Programmer

An AI-powered assistant that helps you write code faster by providing suggestions and tips.

Jupyter Notebooks + AI

Using AI within Jupyter Notebooks for data analysis and code development.

PyCaret

An open-source, low-code machine learning library in Python that helps you quickly build and deploy ML models with minimal code.

TensorFlow

An open-source machine learning framework by Google that allows developers to build and train AI models for a variety of tasks.
